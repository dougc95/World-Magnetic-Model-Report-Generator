wxPython is thinked for Windows, so if you are running it on Linux or WSL you will need to run:
```
sudo apt-get update
sudo apt-get install build-essential gcc python3-dev libgtk-3-dev libwebkit2gtk-4.0-dev libgstreamer-plugins-base1.0-dev libgstreamer1.0-dev
sudo apt-get install libxcb-xinerama0 libxcb-icccm4 libxcb-image0 libxcb-keysyms1 libxcb-randr0 libxcb-render-util0 libxcb-xfixes0 libxcb-shape0
```
